JOYFIT24経堂 — Google Apps Script 構成
========================================

【このリポジトリのファイル】

  オプション出力.gs   … OP・日報連携・JOYFITメニュー・自動トリガー
  入会・退会.gs       … 入会・退会メール・日別反映・退会キャンセル
  退会キャンセル.gs   … 6ヶ月割キャンペーン（取込・在籍期間・一斉送信）
  日報作成用.gs       … 日報メール自動送信・手動下書き
  index.gs            … 旧版（使用しない）

【Google Apps Script への貼り付け】

  運用スプレッドシートのプロジェクトに以下をすべて配置:
    - オプション出力.gs
    - 入会・退会.gs
    - 退会キャンセル.gs
    - 日報作成用.gs

【メニュー JOYFIT】（4項目）

  入会・退会を更新
  6ヶ月割メールを取込（6ヶ月割_入会確認 … 入会メールの台帳）
  キャンペーン対象を整理（在籍期間 … 要連絡者のみを一覧）
  確認メールを送る（送信対象にチェックONの人のみ）

【自動実行】

  19:30 … OP・集計・日報数値・入会退会・当月シート日別欄
  20:00 … 日報メール（土日祝）
  21:00 … 日報メール（平日）
  初回のみ installOptionDailyTrigger を店舗アカウントで実行

【シート一覧】

  日報 / 集計 / OPデータ
  入会・退会 / 入会・退会_データ
  6ヶ月割_入会確認 / 照合 / 在籍期間（要連絡者・送信元）
  入会者（手入力リスト）
  2606 など（月次日報）

【退会キャンセル】

  入会・退会シート J列にチェック → 退会数・日別欄から自動除外
